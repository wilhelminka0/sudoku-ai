 Sudoku AI AssistantTento projekt je moderní webová aplikace pro hraní Sudoku, která integruje umělou inteligenci pro poskytování rad v reálném čase. Aplikace je navržena pro snadné nasazení pomocí Dockeru a využívá perzistentní databázi pro sledování statistik.🚀 Co projekt dělá?Aplikace umožňuje uživatelům generovat a řešit Sudoku různé obtížnosti. Hlavním lákadlem je AI nápověda:Po kliknutí na tlačítko "AI Poradit s polem" odešle aplikace aktuální stav mřížky serveru.Server komunikuje s LLM modelem, který analyzuje situaci a vrátí konkrétní logickou radu v češtině.✨ Klíčové vlastnostiDynamická obtížnost: Volba mezi lehkou, střední a těžkou hrou.Inteligentní rádce: Integrace s modelem gemma3:27b přes API.Perzistentní statistiky: Počítadlo rad se ukládá do databáze, která přežije restart serveru.Plná kontejnerizace: Snadné spuštění kdekoli díky Docker Compose.🛠 Použité technologieČástTechnologieBackendFlask FrameworkFrontendDatabáze(Ukládání v /data)KontejnerizaceDocker ComposeAI ModelGemma 3 (přes Ollama/OpenAI API)📂 Architektura a PerzistenceProjekt je postaven tak, aby splňoval náročné požadavky na moderní vývoj:Docker Compose: Soubor compose.yml definuje dvě služby — aplikaci a databázový uzel, což umožňuje snadnou škálovatelnost.Složka /data: Veškerá uživatelská data (SQLite databáze vlasak.db) jsou ukládána do tohoto adresáře, který je v Dockeru namapován jako volume. To zaručuje, že o své statistiky nepřijdete ani při aktualizaci aplikace.🚦 Jak se spouští (Návod)Příprava prostředí:Ujistěte se, že máte v kořenovém adresáři soubor .env se svými API klíči.Spuštění pomocí Docker Compose:Bashdocker compose up -d --build
Přístup k aplikaci:Otevřete prohlížeč a přejděte na: http://localhost:5000 (nebo na svou iThope URL).📝 Poznámka k vývojiProjekt byl vytvořen jako ukázka propojení webových technologií s lokálně běžícími (nebo cloudovými) jazykovými modely. Klade důraz na čistotu kódu, responzivní design a stabilitu dat.
